# 项目接入规则

任何AI进入本项目必须：

1. 读取 `CORE/memory.md`
2. 读取 `STATE/state.json`
3. 读取 `TASKS/next_actions.json`
4. 读取 `LOGS/change_log.md`

---

## 接入后必须输出：

- 当前项目在做什么
- 当前状态是什么
- 当前下一步任务是什么
- 是否存在风险

---

## 禁止行为：

- 不允许跳过状态读取
- 不允许直接修改代码不更新状态
- 不允许忽略任务优先级
- 不允许跨项目访问其他 brain

---

## 核心原则

1. 每个项目完全独立
2. 不共享任何记忆
3. 所有状态必须写入 STATE
4. 所有任务必须写入 TASKS
5. 所有变更必须写入 LOGS
