# MERCHANT_BRAIN_TEMPLATE

每个商家一个独立目录，一个商家大脑就是一个电子运营官。

## 推荐目录

```text
merchants/{merchant_id}/
├── BOOT.md
├── STATE.json
├── PROFILE.md
├── ACCOUNT_DIAGNOSIS.md
├── STRATEGY.md
├── MATERIALS_INDEX.md
├── CONTENT_LOGS.md
├── PUBLISH_LOGS.md
├── DATA_REVIEW.md
├── TASKS.json
└── DECISIONS.md
```

## 创建前必须具备

- 用户明确授权。
- 商家名称或唯一 ID。
- 账号诊断报告。
- 商家基础资料。

## 隔离规则

- 一个商家目录只保存该商家的记忆。
- 不同商家之间不得混用资料、策略、素材、发布记录或复盘。
- 未授权不得创建真实商家大脑。
