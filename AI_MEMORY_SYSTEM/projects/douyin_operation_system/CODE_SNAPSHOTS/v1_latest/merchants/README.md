# merchants

`merchants/` 是未来商家独立大脑目录。

每个商家一个独立目录，一个商家大脑就是一个电子运营官。

每个商家大脑只保存该商家的：

- 商家档案
- 账号诊断
- 运营方向
- 素材索引
- 内容记录
- 发布记录
- 数据复盘
- 决策记录
- 下一步任务

不同商家之间不得混用记忆。

当前状态：暂不创建真实商家，只创建 `_TEMPLATE/` 模板和规则。创建真实商家大脑必须等待用户明确授权。

## 真实商家大脑创建条件

1. 用户明确授权创建。
2. 已有商家名称或唯一 ID。
3. 已有账号诊断报告，或用户明确允许先建空档。
4. 已有基础商家资料。
5. 必须使用 `_TEMPLATE` 复制生成。
6. 不同商家之间不得复用 `PROFILE`、`STRATEGY`、`PUBLISH_LOGS`、`DATA_REVIEW`、`DECISIONS`。
